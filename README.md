# MBTI 人格测试网站

一个简洁优雅的MBTI人格类型测试网站。

## 功能特性

- ✅ 100道专业测试题（每维度25题）
- ✅ 涵盖MBTI的4个维度（E/I, S/N, T/F, J/P）
- ✅ 实时进度显示
- ✅ 美观的渐变UI设计
- ✅ 详细的16种人格类型描述
- ✅ 维度分析可视化
- ✅ 响应式设计，支持移动端

## 如何使用

### 方式1：本地服务器（推荐）

1. 启动本地服务器：
```bash
cd ~/work/mbti-test
python3 -m http.server 8080
```

2. 在浏览器打开：
```
http://localhost:8080
```

### 方式2：直接打开

直接在浏览器中打开 `index.html` 文件。

## 测试流程

1. **开始页面** - 点击"开始测试"
2. **答题页面** - 回答12道选择题
3. **结果页面** - 查看你的MBTI类型和详细分析

## 技术栈

- HTML5
- CSS3（渐变背景、响应式设计）
- 原生JavaScript（无需框架）
